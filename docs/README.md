# dashboard-LAB Docs

This repository keeps only the shareable boilerplate docs.

- `DASHBOARD_LAB_BLUEPRINT.md`: target product architecture and module boundaries
- `DASHBOARD_LAB_MIGRATION_PLAN.md`: staged work plan for moving from a personal dashboard to a reusable boilerplate
- `ELECTRON_DESKTOP_PLAN.md`: Electron shell, dev workflow, and desktop packaging notes
- `PUBLISHING_CHECKLIST.md`: pre-GitHub and pre-release checklist
- `../CONTRIBUTING.md`: contribution workflow for public collaboration
- `../SECURITY.md`: security reporting and hardening notes

Historical project-specific design notes were intentionally left in the original private workspace.
