# Code of Conduct

## Expectations

Please keep collaboration practical and respectful.

- Assume good intent.
- Critique code and ideas, not people.
- Prefer concrete reproduction steps over vague blame.
- Respect privacy when sharing logs, recordings, or generated documents.

## Unacceptable Behavior

- Harassment or personal attacks
- Publishing private customer or user data
- Intentionally disruptive or misleading contributions

## Enforcement

Maintainers may remove content or contributions that violate these expectations.
